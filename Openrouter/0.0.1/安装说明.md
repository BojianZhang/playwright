# OpenRouter 自动化控制台 — 小白一键安装说明

不用懂代码,照下面做即可。

---

## 一、装之前先准备(只这一个要手动)

**AdsPower 客户端** —— 本工具靠它开浏览器。
1. 去 https://www.adspower.com 下载安装、注册登录。
2. **保持 AdsPower 客户端一直开着**(它自带本地接口,工具靠它干活)。

> Python、Node 这些一键脚本会自动帮你装,不用管。

---

## 二、一键安装

### Windows
**双击 `install.bat`**,然后等它跑完(全程**中文提示**,不用动手)。
- 缺 Python / Node 会**自动用 winget 安装**,装完**当场继续**装依赖 —— 不用关窗口、不用重开。
- 最后会做一次**自检**(验证 selenium 真能用);看到 **「安装完成!」** 就好了。
- 万一自动装失败,它会给你 Python / Node 的官网网址,手动装好后再双击一次即可。

### macOS / Linux(全自动,和 Windows 一样)
打开「终端」,输入 `cd ` (后面带空格),把本文件夹拖进终端窗口(自动填路径),回车;再:
```
bash install.sh
```
- macOS 上**没装 Homebrew 会自动帮你装**(可能要你输一次 **Mac 登录密码**,首次还会装 Xcode 命令行工具,稍慢),然后自动装 Python、Node、依赖 —— **全程自动,不用你手动配环境**。
- Linux 自动走 apt / dnf 装 Python、Node。

---

## 三、一键启动

### Windows:双击 `start.bat`
### macOS/Linux:`bash start.sh`

- 会自动打开浏览器到 **http://localhost:4317**(没自动开就手动输这个网址)。
- 那个**黑窗口就是服务器,别关它**;要停就关掉它。

---

## 四、第一次:填 3 个 key(在网页里填,最简单)

打开 http://localhost:4317 后,进 **「部署引导」或「设置中心」**,填:

| key | 哪来的 | 干啥 |
|---|---|---|
| **Firstmail key** | firstmail.ltd 账户 | 收注册验证邮件 |
| **验证码 key**(capsolver / 2captcha) | capsolver.com 或 2captcha.com 充值后给的 | 过人机验证 |
| **AdsPower key**(如果你的 AdsPower 要) | AdsPower 客户端里 | 连 AdsPower |

> 也可以直接编辑 `config/config.local.json` 文件填(脚本已帮你生成好空模板)。

---

## 五、开始跑

1. 确认 **AdsPower 客户端开着**。
2. 网页进「控制台」→ 选引擎(纯 Selenium)→ 设并发(**建议先 5~8,别太大**)→ 点运行。

---

## 六、(可选)想用「混合 / 分流」引擎(Playwright)

默认的**纯 Selenium 引擎装完就能用,不用看这节**。
如果你要用「混合」或「分流(split)」引擎(它们用 Playwright 开浏览器):
- **Windows**:双击 `install-hybrid.bat`
- **macOS / Linux**:`bash install-hybrid.sh`

它会装 Node 依赖并下载 Playwright 浏览器内核(约 150MB,较慢)。装完到控制台「引擎」里选「混合」或「分流」即可。
> 注:混合引擎需要**完整仓库**;若你只拷了 `Openrouter/0.0.1` 子目录,脚本会提示找不到、自动跳过(不影响纯 Selenium)。

---

## 常见问题

- **双击 install.bat 一闪而过 / 报错**:它会调用 PowerShell 干活;若窗口一闪而过,右键 `install.bat` →「以管理员身份运行」,或把报错截图发出来。
- **没连上 AdsPower**:先打开 AdsPower 客户端并登录,再启动。
- **网页打不开 http://localhost:4317**:确认 `start.bat` 的黑窗口没关、且没报错。
- **号大批「掉线/不动」**:并发调小(高级参数页 `并发硬上限` 设 5~8);**任务跑起来后别重启服务、别手动关浏览器窗口**(会杀掉正在跑的号)。
- **Python 依赖装得慢/失败**:多半是网络,重跑一次;脚本已带国内镜像兜底。
